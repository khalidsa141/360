# 🕌 قصواء مينا - الجولة الافتراضية (نسخة محلية)

## كيفية التشغيل

### الخطوة 1: تحميل جميع الملفات من الخادم
```bash
# استخدام Python (يعمل على Windows/Mac/Linux)
python3 download_assets.py

# أو استخدام Bash (Mac/Linux فقط)
bash download_assets.sh
```
> ⏱️ التحميل يستغرق وقتاً (40 مشهد × 6 وجهات × 4 مستويات = آلاف الصور)

### الخطوة 2: تشغيل الخادم المحلي
```bash
python3 -m http.server 8080
```

### الخطوة 3: فتح المتصفح
```
http://localhost:8080
```

## بنية المجلدات

```
qasswa-local/
├── index.html              ← الملف الرئيسي (بدون أي إشارة لـ madar)
├── download_assets.py      ← سكريبت تحميل الملفات (Python)
├── download_assets.sh      ← سكريبت تحميل الملفات (Bash)
├── tdvplayer.js            ← محرك 3DVista
├── script.js               ← سكريبت الجولة
├── script_general.js       ← إعدادات عامة
├── fonts.css               ← الخطوط
├── favicon.ico
├── fonts/                  ← ملفات الخطوط
├── media/                  ← صور البانوراما (40 مشهد)
│   └── panorama_XXXXX_0/
│       ├── f/0..3/         ← الوجهة الأمامية (4 مستويات تفصيل)
│       ├── b/0..3/         ← الخلفية
│       ├── l/0..3/         ← اليسار
│       ├── r/0..3/         ← اليمين
│       ├── u/0..3/         ← العلوية
│       └── d/0..3/         ← السفلية
├── lib/cursors/            ← مؤشرات الماوس
├── misc/                   ← الأيقونات
├── locale/                 ← ملف اللغة
└── loading/                ← صور شاشة التحميل
```
